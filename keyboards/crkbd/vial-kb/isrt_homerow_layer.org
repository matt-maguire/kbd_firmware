#+author: Matt Maguire

* Corne v4 Layout with Homerow Layer Keys

Another layout inspired by Ben Vallack, this time from his video on the ZSA Voyager. On that keyboard,
he ditched the sticky layers due to the cognitive load needed to keep track of the active layer and
opted for layers selected by holding home row keys (moving these off the thumbs will hopefully avoid
fatigue/pain). The thumb keys are one-shot modifiers. There are also modifiers on the top row except
for "Shift", which can be problematic when performing fast rolls. I also avoid tap-dance keys in the
base layer to minimise false triggers.

This layout consists of 5 layers:
| 0 | Base    |
| 1 | Num+Nav |
| 2 | Sym     |
| 3 | Fun     |
| 4 | Adjust  |

*Note 1*: that the movement keys are in an "inverted-T" arrangement.
*Note 2*: the modifiers on the thumb keys are one-shot.
*Note 3*: I ditched the QWERTY and tap-only layers, as I don't envisage using them.
